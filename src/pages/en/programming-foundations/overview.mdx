## Introduction

In this course we will explore some of the key concepts in computer science, while learning the basics of JavaScript notation. Throughout this module, we will delve into the fundamental principles of computer science, including variables, conditional statements, and arrays, while also gaining a solid foundation in the JavaScript programming language.
